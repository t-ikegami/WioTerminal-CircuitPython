# 天気予報

## 概要
こちらの劣化~~パクリ~~移植。<br/>
https://github.com/99yen/WioTerminal-WeatherSign

Yahoo! の rss に HTTPS 接続して天気情報を取得し、表示します。
地域を変更するには、39 行目の `*.xml` 指定を調整します。
